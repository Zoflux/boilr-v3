# Customer Videos

Place your customer testimonial videos in this folder.

## Supported Formats
- `.mp4` (recommended)
- `.webm`
- `.mov`

## Naming Convention
Name your videos descriptively, e.g.:
- `sarah-mitchell-techscale.mp4`
- `michael-chen-apex-talent.mp4`

## How it works
Videos placed in this folder will automatically be detected and can be displayed on the customer reviews page. The videos are imported in alphabetical order, so name them accordingly if order matters.

## File Size Recommendations
- Keep videos under 50MB for optimal web performance
- Consider compressing videos before uploading
- Recommended resolution: 1080p or 720p
